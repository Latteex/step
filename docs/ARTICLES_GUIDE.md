# Руководство по добавлению статей для проекта «Степной хранитель»

## 📁 Структура

Статьи хранятся в файле: `src/data/articles.ts`

## ➕ Как добавить новую статью

### Шаг 1: Откройте файл

Откройте файл `src/data/articles.ts` в вашем редакторе кода.

### Шаг 2: Скопируйте шаблон

В конце массива `articles` (перед закрывающей квадратной скобкой `];`) добавьте новый объект статьи:

```typescript
{
  id: 7, // Уникальный номер (следующий после последнего)
  icon: "🏷️", // Emoji-иконка для категории
  tag: "Категория", // Название категории (например, "Биология", "Экология", "Питание")
  title: "Заголовок статьи",
  excerpt: "Краткое описание статьи (2-3 предложения). Это текст, который будет виден в превью.",
  content: `
    <h2>Заголовок первого раздела</h2>
    <p>Текст первого параграфа. Здесь можно использовать <strong>жирный текст</strong> и <em>курсив</em>.</p>
    
    <h3>Подзаголовок</h3>
    <p>Ещё текст.</p>
    
    <ul>
      <li>Элемент маркированного списка</li>
      <li>Ещё один элемент</li>
    </ul>
    
    <ol>
      <li>Первый пункт нумерованного списка</li>
      <li>Второй пункт</li>
    </ol>
    
    <blockquote>
      «Цитата от эксперта или важный вывод»
    </blockquote>
  `,
  readTime: "10 мин", // Время чтения
  author: "И.О. Фамилия", // Автор статьи
  date: "1 апреля 2026", // Дата публикации
  image: "https://example.com/image.jpg", // URL изображения для превью
},
```

### Шаг 3: Заполните поля

| Поле | Тип | Обязательное | Описание |
|------|-----|--------------|----------|
| `id` | число | ✅ | Уникальный идентификатор. Используйте следующий номер после последней статьи |
| `icon` | строка | ✅ | Emoji-иконка для категории (🧬, 🗺️, 🍖, 👶, 🛡️, 🔬 и т.д.) |
| `tag` | строка | ✅ | Категория статьи |
| `title` | строка | ✅ | Заголовок статьи |
| `excerpt` | строка | ✅ | Краткое описание (2-3 предложения) |
| `content` | строка | ✅ | HTML-контент статьи (см. ниже доступные теги) |
| `readTime` | строка | ✅ | Время чтения (например, "10 мин") |
| `author` | строка | ✅ | Автор статьи |
| `date` | строка | ✅ | Дата публикации |
| `image` | строка | ✅ | URL изображения для превью |

### Шаг 4: Сохраните файл

После добавления статьи сохраните файл. Изменения автоматически применятся на сайте.

## 📝 Доступные HTML-теги в контенте

В поле `content` можно использовать следующие HTML-теги:

- `<h2>` — Заголовок раздела
- `<h3>` — Подзаголовок
- `<p>` — Параграф текста
- `<strong>` — Жирный текст
- `<em>` — Курсив
- `<ul>` — Маркированный список
- `<ol>` — Нумерованный список
- `<li>` — Элемент списка
- `<blockquote>` — Цитата

### Пример структуры контента

```typescript
content: `
  <h2>Основной раздел</h2>
  <p>Вводный текст раздела.</p>
  
  <h3>Подраздел</h3>
  <p>Текст подраздела с <strong>акцентом</strong> на важном.</p>
  
  <ul>
    <li>Первый пункт</li>
    <li>Второй пункт</li>
    <li>Третий пункт</li>
  </ul>
  
  <h3>Ещё один подраздел</h3>
  <p>Дополнительная информация.</p>
  
  <ol>
    <li>Шаг первый</li>
    <li>Шаг второй</li>
    <li>Шаг третий</li>
  </ol>
  
  <blockquote>
    «Важная цитата от эксперта, которая подчёркивает ключевую мысль.»
  </blockquote>
  
  <p>Заключительный текст.</p>
`,
```

## ✅ Чек-лист перед публикацией

- [ ] `id` уникален и не повторяется с другими статьями
- [ ] Все обязательные поля заполнены
- [ ] В `content` используется только разрешённые HTML-теги
- [ ] URL изображения рабочий и ведёт на изображение
- [ ] Текст в `excerpt` кратко описывает содержание статьи
- [ ] `readTime` реалистично оценивает время чтения
- [ ] Нет опечаток в заголовке и тексте

## 🔧 Технические детали

### Типизация

Все статьи типизированы интерфейсом `Article`:

```typescript
export interface Article {
  id: number;
  icon: string;
  tag: string;
  title: string;
  excerpt: string;
  content: string;
  readTime: string;
  author: string;
  date: string;
  image: string;
}
```

### Где используются статьи

- **Страница статей** (`/articles`) — отображает все статьи в виде сетки карточек
- **Модальное окно** — при клике на статью открывается полное содержание
- **Главная страница** — превью нескольких статей в секции "Научно-популярные материалы"

## ❓ Частые вопросы

### Q: Можно ли добавить изображение внутрь статьи?

A: В текущей версии это не поддерживается. Изображение отображается только в превью и шапке модального окна.

### Q: Можно ли использовать другие Emoji?

A: Да, можно использовать любые Emoji. Рекомендуется выбирать иконку, соответствующую тематике статьи.

### Q: Сколько статей можно добавить?

A: Технически ограничений нет. Однако для удобства навигации рекомендуется не более 20-30 статей.

### Q: Как удалить статью?

A: Удалите соответствующий объект из массива `articles` и сохраните файл.

### Q: Можно ли редактировать существующую статью?

A: Да, просто измените нужные поля в объекте статьи и сохраните файл.
